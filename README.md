# 🎂 Invitación de Cumpleaños

Invitación web personalizada para mi cumpleaños 🎉  

---

## ✨ Features

- 🎨 **Actividad interactiva** — cada invitada elige su frase o diseño para pintar
- 📲 **RSVP vía WhatsApp** — al confirmar asistencia se abre un chat con la info pre-cargada
- 🌙 **Animaciones on scroll** — entradas suaves con Intersection Observer
- 🎊 **Confetti al confirmar** — porque sí
- 📱 **Responsive** — optimizada para compartir por WhatsApp desde el celu

---

| Tecnología | Uso |

| [TypeScript](https://www.typescriptlang.org/) | Tipado |
| [Tailwind CSS](https://tailwindcss.com/) | Estilos |
| [Lucide React](https://lucide.dev/) | Iconos |
| [Vercel Analytics](https://vercel.com/analytics) | Métricas |

---

## 🚀 Cómo correr el proyecto

```bash
# Instalar dependencias
npm install

# Levantar en desarrollo
npm run dev

# Build de producción
npm run build
```

Abre [http://localhost:3000](http://localhost:3000) en el browser.

---

## 📁 Estructura

```
app/
  globals.css       # Variables de tema y estilos base
  layout.tsx        # Metadata y fuentes
  page.tsx          # Página principal
components/
  rsvp-section.tsx  # Formulario + integración WhatsApp
  confetti.tsx      # Animación de confetti
public/
  friends/          # Fotos de amigas
```
